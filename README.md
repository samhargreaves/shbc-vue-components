# VUE Components

Vue componenets library for Stack with:

-   VUE (3)
-   Inertia
-   TailwindCSS
-   Tailwind Elements
-   Laravel (with ziggy routes)

Install with for newest version:

```bash
npm install git@github.com:samhargreaves/shbc-vue-components.git
```
